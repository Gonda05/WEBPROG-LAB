### **🟡 Tahap 4: Master Data (CRUD) (50%)**  
**Tujuan:** Memungkinkan admin untuk mengelola data utama dalam sistem.  
**Task:**  

#### **1️⃣ Master Admin (10%)**  
✅ CRUD untuk data admin (`ID`, `Nama`, `Username`, `Password`, `Email`).  
✅ Admin dapat mengelola pengguna lain (CRUD Users).  

#### **2️⃣ Master Dosen (10%)**  
✅ CRUD untuk data dosen (`NIK`, `Nama`, `Gelar`, `Lulusan`, `Telp`).  

#### **3️⃣ Master Mahasiswa (10%)**  
✅ CRUD untuk data mahasiswa (`NIM`, `Nama`, `Tahun Masuk`, `Alamat`, `Telp`).  

#### **4️⃣ Master Mata Kuliah (10%)**  
✅ CRUD untuk data mata kuliah (`Kode Matkul`, `Nama Matkul`, `SKS`, `Semester`).  

#### **5️⃣ Manajemen Pengguna (10%)**  
✅ Admin dapat menambahkan, mengedit, menghapus pengguna lain (`Admin`, `Dosen`, `Mahasiswa`).  
✅ Pastikan setiap user memiliki role yang sesuai.  

💡 **Tips:** Kerjakan Master Admin lebih dulu karena berperan dalam mengelola user lain.
